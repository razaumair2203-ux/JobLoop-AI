Development vs Production Handoff

DEVELOPMENT

  • LLM: Claude Code (you) IS the LLM. Zero API calls.
  • Flow: User uploads → Supabase stores extracted text → Claude Code reads, parses, writes JSON back
  • Extraction: pdf.js-extract (local, no API)
  • Auth: DEV_AUTH_BYPASS=true
  • Cost: $0 (Claude Code subscription only)
  • Steps 1-4, 6-8, 10-12: Identical to production (pure code)
  • Steps 5, 9: Claude Code replaces DeepSeek API

PRODUCTION

  • LLM: DeepSeek Flash API (deepseek-chat)
  • Flow: User uploads → extract → DeepSeek parses → Cloud builds automatically
  • Extraction: pdf.js-extract (same as dev)
  • Auth: Supabase Auth (Google + LinkedIn OAuth)
  • Cost: ~$0.001/CV parse, ~$0.007/JD analysis (DeepSeek V4 Flash)
  • Handoff: Set DEEPSEEK_API_KEY env var. That's it.
  • Same code: Zero code changes between dev and prod
