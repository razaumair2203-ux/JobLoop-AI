# AI-Native CV/Resume Building Tools: Deep Dive Research (May 2026)

Research conducted via live web searches. All capabilities verified against product pages, help docs, user reviews, and third-party walkthroughs. NO fabricated capabilities.

---

## 1. TEAL (tealhq.com)

**What it actually does:**
- **Master Resume**: Yes. A comprehensive "source document" containing your full work history. Not meant to be sent — you pull from it.
- **Resume Syncing**: New bullet points added to one resume are instantly available in all versions. "Save to all resumes" toggle. "Update available" flags across versions.
- **AI Tailoring**: Attach a JD to any resume → get a match score → AI suggests keywords, toggles bullets on/off for relevance. One click.
- **Job Tracker**: Kanban board. Chrome extension saves jobs from any website.
- **Match Score**: Compares resume vs JD. Reported at ~78% accuracy (lower than Jobscan's ~87% in one comparison).
- **Skill Gap Analysis**: Compares your profile to job requirements, highlights gaps.
- **Pricing**: Free tier (unlimited resumes + tracking). Teal+ $29/mo or $9/week.

**Answering the key questions:**
| Question | Answer |
|----------|--------|
| Multiple CVs from same person? | YES — master resume + unlimited tailored versions, all synced |
| Conflict/inconsistency detection? | NO — no evidence of this capability |
| Persistent skill profile that evolves? | PARTIAL — master resume acts as persistent source, but it's a document, not a structured profile |
| Follow-up questioning to enrich? | NO — no Socratic/conversational enrichment |
| Skills linked to evidence? | NO — skills are a flat list you toggle on/off. The platform *advises* linking skills to bullets, but doesn't enforce or automate it |
| Data quality gating? | NO — no blocking on data quality |

**User flow**: Sign up → Import LinkedIn or upload resume → Edit in builder → Save as master → For each job: attach JD → view match score → toggle bullets → AI suggests keywords → download tailored version.

**Honest assessment**: Best-in-class for the "master resume + tailored versions" workflow. The syncing feature is genuinely useful. But it's a document manager with keyword matching, not a profile intelligence system. Skills are flat lists. No depth, no evidence linking, no questioning.

## 2. REZI (rezi.ai)

**What it actually does:**
- **AI Resume Agent**: Conversational AI that reads your resume context and can tailor, improve, and optimize. You can ask follow-up questions and request changes — it's a chat interface.
- **AI Skills Explorer**: Pick a category and skill from your line of work → Rezi suggests related/transferable skills to add. Click to add.
- **Rezi Score**: 1-100 score across 23 criteria (bullet length, active verbs, quantification, filler language, formatting). Updates in real-time as you edit. Five core categories, 20+ individual audits.
- **Work Experience Assistant**: AI chatbot that asks follow-up questions about your experience and generates action-driven bullet points. Prompts you to add metrics, outcomes, scope.
- **Real-Time Analysis**: Updates within ~5 seconds. Red = high-impact formatting/keyword errors. Yellow = content enhancements.
- **Pricing**: Free plan, Basic $3/mo, Pro $29/mo, Lifetime $129.

**Answering the key questions:**
| Question | Answer |
|----------|--------|
| Multiple CVs from same person? | YES — document version management, multiple resumes |
| Conflict/inconsistency detection? | NO — checks formatting/quality, not logical consistency |
| Persistent skill profile that evolves? | NO — skills are per-resume, not a unified profile |
| Follow-up questioning to enrich? | PARTIAL — Work Experience Assistant asks follow-ups about bullets. AI Chat acts as resume coach prompting for metrics/outcomes. But this is per-bullet, not profile-level |
| Skills linked to evidence? | NO — Skills Explorer suggests skills, but doesn't link them to roles/duration/impact |
| Data quality gating? | PARTIAL — Rezi Score flags issues, but doesn't block/gate. Advisory only |

**User flow**: Sign up → Upload resume, import LinkedIn, or describe background → AI Agent reads context → Edit in builder → Real-time score updates → Use Skills Explorer to find gaps → Work Experience Assistant for bullet writing → Tailor to JD → Download.

**Honest assessment**: Rezi's Work Experience Assistant is the closest any competitor gets to "follow-up questioning" — but it's bullet-level, not profile-level. The Skills Explorer is a suggestion engine, not a depth classifier. The 23-point score is structural (formatting, length, verbs), not semantic (does this skill match this evidence?). The AI Agent is a chat wrapper, not a proactive enrichment engine.

## 3. JOBSCAN (jobscan.co)

**What it actually does:**
- **Match Rate**: Paste resume + JD → get a percentage match. Breaks down into Searchability, Hard Skills, Soft Skills, Formatting. Recommends 75-80%.
- **Power Edit**: Inline editing with scan results alongside. Edit resume while seeing which keywords you're missing.
- **Predicted Skills**: AI predicts skills a hiring manager might look for BEYOND what's in the JD — based on similar JDs. You can report irrelevant predictions.
- **ATS Detection**: Enter company name → Jobscan identifies which ATS they use (iCIMS, Greenhouse, Lever, Taleo, Workday) → tailors advice accordingly.
- **Multi-ATS Simulation**: Models how different ATS platforms would parse and score your resume differently. This is unique.
- **30+ Parameters**: Formatting, word count, measurable results, keyword density, education level, job title match.
- **Pricing**: $49.95/mo monthly, $89.95/3 months (~$30/mo). No annual.

**Answering the key questions:**
| Question | Answer |
|----------|--------|
| Multiple CVs from same person? | NO evidence of master resume or multi-version management |
| Conflict/inconsistency detection? | NO — purely keyword/format matching |
| Persistent skill profile that evolves? | NO — each scan is job-specific, no persistent profile |
| Follow-up questioning to enrich? | NO — no conversational element |
| Skills linked to evidence? | NO — skills are keyword presence/absence checks |
| Data quality gating? | NO — advisory match score, no blocking |

**User flow**: Paste resume text + paste JD → view Match Rate breakdown → see missing hard skills, soft skills → use Power Edit to add keywords inline → re-scan → see Predicted Skills → optionally use resume builder → download.

**Honest assessment**: The strongest keyword-matching engine. Predicted Skills is genuinely interesting — it's the closest anyone gets to "enrichment thinking" beyond the JD. Multi-ATS simulation is unique and valuable. But it's fundamentally a keyword diff tool. No depth, no profile, no intelligence beyond keyword matching. Expensive at $50/mo for what it is.

No intelligence beyond text generation. The GPT-5 claim is marketing; the output is generic bullet points. Good templates, zero depth.

## 8. LOOPCV (loopcv.pro)

**What it actually does:**
- **Auto-Apply Pipeline**: Upload resume + set criteria (titles, location, keywords, seniority) → LoopCV scans job boards → auto-applies on your behalf OR notifies you (semi-auto mode).
- **Multiple CV Upload**: Upload different resumes, assign to different "Loops" (search configurations).
- **A/B Testing**: The UNIQUE feature — test different resume versions, track which gets better response rates. Metrics: email open rates, reply rates, interview conversion.
- **Email Outreach**: Automated outreach emails to HR/recruiters with your CV attached.
- **Smart Duplicate Prevention**: Won't reapply to same company/job across boards.
- **CV Checker**: ATS compatibility analysis.
- **Runs 24/7**: Server-side, works when your computer is off.
- **Pricing**: Paid plans, lifetime deals available on StackSocial.

**Answering the key questions:**
| Question | Answer |
|----------|--------|
| Multiple CVs from same person? | YES — multiple CVs assigned to different Loops |
| Conflict/inconsistency detection? | NO |
| Persistent skill profile that evolves? | NO — your profile is your uploaded CV, not a structured intelligence |
| Follow-up questioning to enrich? | NO |
| Skills linked to evidence? | NO |
| Data quality gating? | NO — will submit whatever you upload |

**User flow**: Sign up → Upload CV(s) → Set search criteria per Loop → Choose full-auto or semi-auto → LoopCV scans boards 24/7 → Auto-applies or notifies → Track open rates, reply rates → A/B test CV variants → Iterate.

**Honest assessment**: LoopCV is NOT a CV builder — it's a CV distribution/automation engine. The A/B testing on resume response rates is genuinely unique and valuable. Nobody else lets you empirically test which resume version gets more callbacks. But it treats your CV as a blob to submit, not something to understand. The same untailored CV goes to every matched job (though you can assign different CVs to different Loops). Application-to-response ratio tracking (~2-3% baseline) is real data most tools ignore.

## 9. HUNTR (huntr.co)

- **Job Tracker**: Drag-and-drop application management.
- **Pricing**: Free tier, paid for advanced AI features. 400K+ users.

**Answering the key questions:**
| Question | Answer |
|----------|--------|
| Multiple CVs from same person? | YES — one (or multiple) base resumes + unlimited tailored versions per job |
| Conflict/inconsistency detection? | NO |
| Persistent skill profile that evolves? | PARTIAL — base resume persists but is a document, not a structured profile |
| Follow-up questioning to enrich? | NO |
| Skills linked to evidence? | NO — keyword matching only |
| Data quality gating? | NO — advisory scores only |

**Honest assessment**: Very similar architecture to Teal and Careerflow (base + tailored clones). The level-aware review (junior vs executive vocabulary) is a nice touch nobody else does explicitly. Content Length Score is minor but thoughtful. Fundamentally the same keyword-matching paradigm as everyone else.

## KEY FINDINGS FOR JOBLOOP

### What NOBODY does (our exploitable gaps):
1. **Conflict/inconsistency detection** — ZERO competitors detect date overlaps, title mismatches, or contradictory data across CVs. Not one.
2. **Persistent evolving skill profile** — Everyone has "resumes," nobody has a "Profile Cloud" that grows over time. The closest is a master resume document (Teal, Huntr, Careerflow), but those are documents, not structured skill graphs.
3. **Evidence-linked skills** — ZERO competitors connect skills to specific roles, durations, or impact metrics programmatically. Rezi's Skills Explorer suggests skills; it doesn't validate them against your history.
4. **Follow-up questioning for profile enrichment** — Rezi's Work Experience Assistant asks bullet-level questions (closest competitor). Kickresume has a basic questionnaire. Nobody does profile-level Socratic enrichment.
5. **Data quality gating** — Nobody blocks or gates anything based on data quality. Every tool will happily process garbage input.
6. **Multi-CV merging into unified profile** — Nobody merges multiple CVs into a single structured representation. Teal syncs content across versions but doesn't merge/deduplicate.
7. **Outcome-based learning** — Nobody feeds application outcomes back into the profile or CV generation. LoopCV's A/B testing is the closest (empirical feedback on which CV version works better), but it doesn't feed into content generation.

### What competitors do well (learn from):
1. **Teal's resume syncing** — Genuinely useful UX for managing multiple versions. "Save to all resumes" toggle is elegant.
2. **Jobscan's multi-ATS simulation** — Knowing how iCIMS vs Greenhouse vs Lever would parse your resume differently is valuable intelligence.
3. **Jobscan's Predicted Skills** — Going beyond the JD to suggest skills commonly found in similar roles. This is enrichment thinking.
4. **Rezi's Work Experience Assistant** — Follow-up questions for better bullets. Limited but points in the right direction.
5. **LoopCV's A/B testing** — Empirical resume testing is the only real-world feedback loop in this space. Track what actually works.
6. **Huntr's level-aware content** — Junior vs executive vocabulary modulation. Nobody else does this explicitly.
7. **Careerflow's LinkedIn Chrome overlay** — Real-time scoring directly on LinkedIn profile page.

### What's mostly marketing:
1. **Kickresume's "GPT-5 powered"** — The output is generic bullet points regardless of model.
2. **Sonara's "70% interview rate"** — Unverifiable, platform is now dead.
3. **Rezi's "23 criteria"** — Most are formatting checks (font size, file type, bullet length). Structural, not semantic.
4. **Match scores everywhere** — Every tool gives a percentage. None explain what it means or whether it correlates with outcomes. Jobscan's is the most defensible because it's keyword-specific.
5. **"ATS-friendly" templates** — Every tool claims this. The reality is that single-column, standard-header PDFs work. The template itself is not the differentiator.

## BOTTOM LINE

The entire market is stuck at **keyword matching + template selection + match score**. That's the ceiling. The most advanced thing any tool does is:
- Teal: sync bullet points across resume versions
- Jobscan: simulate different ATS parsing
- Rezi: ask follow-up questions about individual bullets
- LoopCV: A/B test which resume gets more callbacks

Nobody — not a single tool — builds a structured, evidence-based skill profile from your data. Nobody detects conflicts. Nobody asks why your title changed. Nobody remembers what worked in your last application and uses it for the next one. Nobody gates on data quality.

The gap is real. The gap is wide. And nobody appears to be closing it.
